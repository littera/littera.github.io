Installation

Server Requirements

  • PHP >= 5.5.9
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension

Installing Littera

Via Composer Create-Project

$ composer create-project littera/littera your-project-name --prefer-dist
$ cd you-project-name
$ npm install
$ bower install
$ gulp
